Welcome to Syllahub

Syllahub is a one stop syllabus managment system for Kent State University! Here instructors can create syllabi for the courses they teach and students can search for courses and view the syllabi for them. If you are an instructor log in with the account provided by an administrator. Otherwise you can search for a course by typing a course number in the navbar above.
